# Copilot Instructions Snapshot

This repository follows the project requirements for **ISS-Horizon**:

- Predict ISS windows for a chosen location.
- Generate monthly grouped reports.
- Optionally send reports via SMTP.
- Support systemd timer deployment for Debian 12 LXC.
- Keep documentation, docstrings, code comments, and user-facing help text in English.

Authoritative instructions are maintained in the original planning document used during project setup.
